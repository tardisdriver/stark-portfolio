Projects

SHRUB - The tech support chat bot

Live Demo Check this out on GitHub

NOTE: This app is currently under active development. If it is not functioning, please check back later.

SHRUB stands for Support Helper Utility Responsive Bot. She is intended to sit between consumers and human tech support for a computer company and handle frequently asked questions in an intelligent way. She uses IBM Watson Conversation and Discovery APIs, and thus is still currently learning how to do her job!
Stack used: ReactJS, NodeJS, MongoDB, Watson APIs

Travel Buddy - Budget your travel expenses

Live Demo Check this out on GitHub
Travel Buddy easily stores your travel budget information locally through your browser, or if you'd like to access your trip information on multiple devices, you can also save your progress to the server.
Stack used: HTML, CSS, JS, NodeJS, MongoDB

Dinner and a Beer - Perfect beer pairings for random recipes

Live Demo Check this out on GitHub
Dinner and a Beer allows the user to select which protein/entree type they'd like, and the app will suggest a random beer in a style that matches the recipe. Perfect for indecisive beer lovers looking to try new brews.
Stack used: HTML, CSS, jQuery, Edamam API, BreweryDB API
